# OYKZINE テーマ（土台 v0.1.0）

紙のZINE「OYKZINE」のWeb版テーマの出発点です。
このバージョンには **コンテンツの型（号 / Notes / 作品）** と
**メニューの枠** が入っています。デザインはまだ入っていません。

## このフォルダの中身

| ファイル | 役割 |
|---|---|
| `style.css` | テーマ情報＋最低限のリセットCSS |
| `functions.php` | 投稿タイプ・連載・メニューの登録（テーマの心臓部） |
| `header.php` / `footer.php` | ページの上下の最小限の枠 |
| `index.php` | テンプレートを作るまでの予備表示用 |

## 入れ方（ローカル環境ができている前提）

1. このフォルダ `oykzine` ごと `wp-content/themes/` の中に置く
   （または管理画面 外観 > テーマ > 新規追加 > テーマのアップロード から zip を入れる）
2. 管理画面 **外観 > テーマ** で「OYKZINE」を有効化する
3. **設定 > パーマリンク** を開いて、何も変えずに「変更を保存」を押す
   - これをやらないと `/issues/` などのURLが 404 になります（毎回ではなく一度でOK）

## 有効化したらこうなる

- 管理画面の左メニューに **号(Issues) / Notes / 作品(Works)** が追加されます
- 号やNoteを編集する画面に「連載」を付けられるようになります
- **外観 > メニュー** に「フッターナビ」「Menuの中身」の2つの置き場所が出ます

まずは試しに号を1つ、Noteを1つ追加して、表紙画像（アイキャッチ）を設定してみてください。
予備の `index.php` でタイトルが表示されれば土台はOKです。

## 次のステップ（まだやっていないこと）

1. **ACF（Advanced Custom Fields）** を入れて、号に
   「VOL番号」「リード文」「アクセスコード」などのフィールドを足す
2. `front-page.php` を作って、流れる表紙＋最新号ヒーロー＋Notesフィードを組む
3. `single-issue.php` などの個別ページと、凝った記事テンプレ（部品）を作る
4. 下部ナビ（Home / Search / Menu）の見た目を作る

ここから先は一緒に1つずつ作っていきます。
